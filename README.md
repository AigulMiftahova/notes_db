# notes_db
This repository contains the database schema for a personal notes management system. It includes tables for managing users, folders, tags, notes, and many-to-many relationships between notes and tags.

⭐Database Structure
The database consists of the following tables:
· Users – registered users
· Folders – user‑owned folders for organizing notes
· Tags – unique tag names
· Notes – notes with title, content, pin status, and timestamps
· note_tags – many‑to‑many relationship between notes and tags
